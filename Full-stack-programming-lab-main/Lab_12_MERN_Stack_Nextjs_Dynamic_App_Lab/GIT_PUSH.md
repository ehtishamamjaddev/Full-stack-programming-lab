Git push helper
----------------

Run these commands from the repo root to prepare and push the `Lab_12_MERN_Stack_Nextjs_Dynamic_App_Lab` folder to GitHub. Replace `origin` and `main` as required for your remote branch.

```bash
git add .
git commit -m "Complete Lab 12: MERN + Next.js app — admin UI and shadcn-style components"
git push origin main
```

If you want to push only the lab folder:

```bash
cd Lab_12_MERN_Stack-Nextjs_Dynamic_App_Lab
git init
git remote add origin <your-repo-url>
git add .
git commit -m "Initial lab 12"
git push -u origin main
```

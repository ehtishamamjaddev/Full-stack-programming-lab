# Counter App

## Purpose
This app demonstrates React state management by updating a visible counter in real time.

## Concepts Covered
- Functional components
- useState hook
- Conditional button disabling

## Features
- Current count display
- Increment button (+1)
- Decrement button (-1) with floor at 0
- Reset button
- Professional dark theme card layout

## How to Run
1. npm install
2. npm start

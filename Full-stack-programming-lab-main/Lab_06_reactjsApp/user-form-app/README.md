# User Form App

## Purpose
This app demonstrates controlled form handling using React state and event handling.

## Concepts Covered
- Controlled inputs
- useState for form data and errors
- Event handling with onChange and onSubmit

## Features
- Name and email inputs managed by state
- Input validation:
	- Name required
	- Email must include @
- Inline error messages
- Success message after valid submission
- Submitted data preview
- Form reset after successful submission

## How to Run
1. npm install
2. npm start

## Task Descriptions

### Task 1 – Dynamic List Manager
A dynamic list application where users can add items via an input field, delete individual items with smooth animations, and see hover highlight effects. Demonstrates DOM manipulation, event handling, and CSS manipulation using jQuery.

### Task 2 – Animated Image Gallery
An image gallery that displays one image at a time with Next/Previous navigation. Uses jQuery `fadeIn()` and `fadeOut()` transitions with dynamic caption updates and proper chaining for smooth, elegant animations.

### Task 3 – Interactive Form Validation
A professional form with Name, Email, and Password fields. Validates on blur with inline error messages, highlights invalid fields, prevents page refresh, and displays a dynamic success message upon valid submission.

### Task 4 – Tabbed Content with Smooth Scroll
A tabbed interface with 5 content sections. Clicking a tab reveals its associated content with smooth animations while hiding other sections. Includes smooth scroll to the active content area.

### Task 5 – Chained Style Editor
A text styling tool with controls for font size, text color, background color, bold toggle, and italic toggle. Utilizes jQuery chaining to apply multiple style changes simultaneously to a text block.

### Task 6 – API Data Fetcher
Fetches posts from the JSONPlaceholder API using jQuery AJAX. Initially loads 5 posts in a clean card layout with a "Load More" button, a loading indicator, and proper error handling.

### Task 7 – Drag and Drop Sortable List
A sortable list using jQuery UI where items can be dragged and reordered. Highlights items while dragging, updates the order dynamically, and displays the current order below the list.

### Task 8 – Quiz Game
An interactive multiple-choice quiz that presents one question at a time with a Next button, tracks the user's score, reveals the final result with animation, and provides a reset option.

## How to Run Locally

1. Clone or download this repository.
2. Navigate to the `Lab_04_ES6_JS` folder.
3. Open any task's `index.html` file directly in a web browser.
4. Each task works independently — no build tools or server required.
5. An active internet connection is needed for the jQuery CDN and API calls (Task 6).

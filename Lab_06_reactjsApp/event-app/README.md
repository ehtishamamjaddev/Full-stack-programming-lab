# Event App

## Purpose
This app demonstrates React event handling with click and hover interactions.

## Concepts Covered
- onClick event
- onMouseOver and onMouseOut events
- State updates from user interactions

## Features
- Show Message button
- Change Background button that cycles dark background colors
- Show Alert button using alert()
- Hover color feedback on heading
- Smooth background transition and message fade-in animation

## How to Run
1. npm install
2. npm start

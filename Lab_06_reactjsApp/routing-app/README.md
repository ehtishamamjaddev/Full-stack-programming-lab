# Routing App

## Purpose
This app demonstrates client-side routing using React Router and multiple pages.

## Concepts Covered
- BrowserRouter setup
- Routes and Route mapping
- Link-based navigation
- Form events and validation

## Features
- Pages: Home, About, Contact, Products, NotFound
- About page with team members:
	- Ehtisham Amjad
	- Maryam Noor
	- Shareef Hussain
- Contact form with basic validation and success message
- Product cards with Add to Cart buttons
- 404 page and link back to Home

## How to Run
1. npm install
2. npm start
